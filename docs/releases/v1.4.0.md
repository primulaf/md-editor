# v1.4.0

发布日期：2026-07-15

v1.4.0 聚焦长文档导航、多文档辨识和阅读可访问性，并重新分配侧栏空间，使目录成为主要工作区域。

## 新功能

- 目录改为可折叠树，支持逐级展开和收起。
- 当前章节位于折叠分支时自动展开父级，并滚入目录可视区域。
- 字号由三档扩展为小、中、大、特大、超大五档，并保留用户选择。
- 浏览器标签页显示当前文件名；未保存时增加 `*` 标记。

## 界面优化

- 侧栏文档操作改为紧凑的三列两行工具区。
- 删除重复的功能介绍区域。
- 统计信息改为底部四列摘要，为目录释放主要空间。
- 在 1280 × 720、超大字号下，目录可视高度由约 91px 提升至 389px。

## 修复与性能

- 修复只修改正文后标题锚点丢失、目录偶发点击无反馈的问题。
- 修复空白文档刷新后被示例内容替换的问题。
- 重复标题现在会获得稳定且唯一的锚点。
- 目录点击会同时定位编辑区和预览区。
- 目录高亮使用节点映射更新，减少长目录滚动时的重复遍历。

## 验证

- 五档字号循环与本地偏好恢复通过。
- 目录折叠、跳转、高亮、自动展开与自动滚动通过。
- 空白文档、重复标题和正文重渲染场景通过。
- 1280 × 720 桌面端及 800 × 900 窄屏视觉回归通过。
- `npm test`、Manifest 解析和浏览器控制台检查通过。

## 安装

下载 Release 附件 `md-editor-extension.zip`，解压后在 `chrome://extensions` 中开启开发者模式并选择“加载已解压的扩展程序”。直接打开本地 Markdown 文件前，请在扩展详情中开启“允许访问文件网址”。
